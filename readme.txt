■requirements
・ruby 1.9.2
・RMagick 2.13.1

■usage
typing command as follows. 
$ ruby unshredder.rb [Shredded image filename]

then creating unshredded image at the same directory.

ex)
ruby unshredder.rb TokyoPanoramaShredded.png
-> TokyoPanoramaShredded_unshredded.png
